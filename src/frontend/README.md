# Frontend (React + Vite)

Simple UI for:
- Uploading videos
- Creating cameras from videos
- Starting/stopping/restarting cameras
- Copying `/dev/videoX` device paths

## Configure API base URL

Create `src/frontend/.env` (optional):

```bash
VITE_API_BASE_URL=http://localhost:8000
```

## Install

From `virtual-camera-platform/src/frontend`:

```bash
npm install
```

## Run

From `virtual-camera-platform/src/frontend`:

```bash
npm run dev
```
